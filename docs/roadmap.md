# AnyDrop Roadmap

Project: AnyDrop

Tagline:

"AirDrop for Everyone"

---

# Vision

AnyDrop aims to become a universal file-sharing platform that works across:

* Windows
* Linux
* macOS
* Android
* iPhone
* Tablets
* Smart TVs

using only a web browser.

---

# Core Principles

1. No account required
2. No app installation required
3. Direct device-to-device transfer
4. Fast local transfers
5. Cross-platform compatibility
6. Privacy-first design

---

# Phase 1 — MVP

Status: In Development

Goal:

Transfer files between two devices on the same local network.

Features:

* Device discovery
* Device naming
* Transfer requests
* Accept / decline transfers
* WebRTC DataChannel
* File metadata exchange
* Chunked file transfer
* Download reconstruction

Success Criteria:

* Windows ↔ Windows
* Android ↔ Android
* Windows ↔ Android
